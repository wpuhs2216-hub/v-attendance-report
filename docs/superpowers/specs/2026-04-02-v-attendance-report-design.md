# V出勤報告ジェネレーター 設計書

## 概要

ホストクラブ「V」の日次出勤報告テンプレートを生成し、コピペしやすく出力するWebスマホアプリ。

## 技術スタック

- Vanilla JS + Vite
- データ保存: localStorage
- デプロイ: ローカル開発サーバー or 静的ホスティング

## 画面構成（1画面完結）

### 1. 日付セクション
- 今日の日付を自動セット（曜日付き）
- 日付変更可能（date input）

### 2. 人数のみ入力セクション（名前選択なし）
以下の4項目は数値入力のみ:
- オープンメンバー
- スライド
- 体験者
- 仮入店者

### 3. 名前振り分けセクション（名前選択あり）
以下の10項目は登録済み名前一覧からタップで選択:
- 同伴
- シフト出勤者
- 新規入店者
- 遅刻
- 欠勤
- シフト欠
- 研修
- 不明
- 公休
- 休業

**振り分けUI:**
- 未割り当ての名前一覧を表示
- 名前タップ → カテゴリ選択 → 振り分け完了
- 振り分け済みの名前はカテゴリ下に表示（タップで解除）
- 人数は自動カウント

### 4. 出力セクション
- テンプレート形式でリアルタイムプレビュー
- 「コピー」ボタンでクリップボードにコピー
- コピー完了時にフィードバック表示

### 5. 名前管理（設定）
- 名前の追加・削除
- localStorage に永続化

## テンプレート出力形式

```
V出勤報告 {月}月{日}日（{曜日}）
オープンメンバー{N}人
スライド {N}人
同伴　{N}人　{名前1}　{名前2}...

体験者　{N}人

シフト出勤者　{N}人　{名前...}
新規入店者　　{N}人　{名前...}
仮入店者　{N}人
遲刻　{N}人　{名前...}
欠勤  {N}人 {名前...}
シフト欠　{N}人 {名前...}
研修　{N}人　{名前...}
不明　{N}人　{名前...}
公休　{N}人　{名前...}
休業　{N}人　{名前...}

在籍人数 {N}名

以上宜しくお願い致します。
```

## 在籍人数の計算

在籍人数 = オープンメンバー + スライド + 同伴 + 遅刻 + 欠勤 + シフト欠 + 研修 + 不明 + 公休 + 休業

**除外項目:** 体験者、シフト出勤者、新規入店者、仮入店者
